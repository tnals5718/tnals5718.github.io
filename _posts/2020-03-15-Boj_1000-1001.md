## 백준 알고리즘 > 1000, 1001

#### 1000
##### 문제 : 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.
``` c
#include <stdio.h>
int main() {
    int a,b;

    scanf("%d %d", &a, &b);
    printf("%d", a+b);

    return 0;
}
```

#include <stdio.h>
